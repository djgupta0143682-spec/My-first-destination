# My Dream Destination
live link/https://djgupta0143682-spec.github.io/My-first-destination/
This project is a simple website built around `index.html` and a small asset folder.

## Files

- `index.html` - Main homepage for the site.
- `assets/CSS/style.css` - Styles used by `index.html`.
- `assets/image/tajmahel.avif` - Featured image.
- `public/contact.html` - Optional contact page.

## How to view

1. Open `index.html` in your browser.
2. If you have a local web server, point it to this project folder and open `/index.html`.

## Notes

- `index.html` is the main entry point of this project.
- The page uses the CSS file in `assets/CSS/style.css`.
- The `public/contact.html` file can be used for a contact page or additional navigation.
